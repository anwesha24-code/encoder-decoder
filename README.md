## ğŸ§  How It Works  

The app mimics how text messages were typed on **classic mobile keypads**.  
Each number corresponds to a set of letters:  

| Key | Letters |
|:----|:---------|
| 2 | A, B, C |
| 3 | D, E, F |
| 4 | G, H, I |
| 5 | J, K, L |
| 6 | M, N, O |
| 7 | P, Q, R, S |
| 8 | T, U, V |
| 9 | W, X, Y, Z |

When encoding, letters are converted to their respective number sequences.  
When decoding, the number sequences are turned back into letters â€” just like texting before smartphones!
